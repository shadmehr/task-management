Welcome to the TODO / Task Management! A simple SPA example, built with:

  • ASP.NET Core 3.1 and C# runing on Azure AppService(linux-based)
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling
  • Azure SQL Server 2019 for persiting data

Architecture:

  • DDD CQRS without Aggregation pattern.
  • Data Base Migration with EF 5.
  • Repository Pattern.
  • Swagger API Documention.
