Welcome to the TODO / Task Management! A simple SPA example, built with:

  • ASP.NET Core 3.1 and C# runing on Azure AppService (Linux-based).
  • Angular and TypeScript for client-side code.
  • Bootstrap for layout and styling.
  • Azure SQL Server 2019 for persisting data.

Architecture Á Technologies:

  • Light Version of DDD CQRS without Aggregation pattern.
  • Data Migration with EF 5.
  • Repository Pattern.
  • Mediat Pattern (MediatR).
  • Swagger.
  • Automapper.
  • FluentValidation.
